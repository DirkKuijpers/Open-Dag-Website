# Open Dag Website – Curio

Een informatieve website gemaakt voor de open dag van Curio. Bezoekers kunnen info vinden over de school, docenten, lessen, locatie en mijn persoonlijke ervaring.

## Live Demo
[open-dag-website-production.up.railway.app](https://open-dag-website-production.up.railway.app/)

## Pagina's
- **Home** – algemene info over de open dag
- **Docenten** – overzicht van docenten
- **Lessen** – info over de lessen
- **Locatie** – waar de school zich bevindt
- **Ervaring** – mijn persoonlijke ervaring

## Talen
- HTML
- CSS
- JavaScript
